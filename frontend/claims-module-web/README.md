# Claims Module Web

Angular frontend shell for the DICEUS Claims Module assessment.

## Local Development

Install dependencies:

```bash
npm install
```

Start the frontend:

```bash
npm start
```

Open `http://localhost:4200`.

The backend API base URL is configured in `src/environments/environment.ts` and currently points to `http://localhost:5188`.

## Build

```bash
npm run build
```

## Test

```bash
npm test
```
